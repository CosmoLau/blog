使现代的原生 Git Hooks 变得简单

Husky 能使你的提交变得更好 🐶 汪!

在提交或推送时，你可以用它来 检查提交信息、运行测试、检查代码 等。Husky 支持 所有 Git hooks。

点击 这里 开始使用。

特点 ​

  • 轻量级，零依赖 (6 kB)
  • 由现代化的新 Git 特性驱动 (core.hooksPath)
  • 遵循 npm 和 Yarn 的 自动安装 最佳实践
  • 对用户友好的信息
  • Opt-in/opt-out
  • 支持
    • macOS, Linux 和 Windows
    • Git GUIs
    • 自定义 hooks 目录
    • 嵌套项目
    • Monorepos
